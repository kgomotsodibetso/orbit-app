01
Gradient Body
Lavender #C4C0FB → Steel #4B8EBA → Ink #1C2830 — diagonal 135°. Never replace with flat color.
02
Orbit Arc
White → Golden gradient ellipse at −40° rotation. Represents orbital motion. Stroke weight: 7.5px at 160px canvas.
03
Golden Dot
#F6B93B, 9.5px radius. Always sits at the arc tip — top right quadrant. It's the anchor and the accent.
04
Clear Space
Minimum clear space = ½ the mark radius on all sides. Never place competing marks within this zone.
05
Minimum Size
Mark only: 24px. Horizontal lockup: 32px mark height. Below 24px, use favicon-optimised flat version.

07 — Usage Rules

Do's & Don'ts

✓
Do
✓
Always use the gradient body — Lavender → Steel → Ink at 135°. The mark only works with this gradient.
✓
Use the mark at minimum 24px. Below 24px, use the simplified favicon version without the arc detail.
✓
Pair the mark with Montserrat 900 for the wordmark. Maintain −1px letter-spacing at 24px+.
✓
Use Golden #F6B93B exclusively for the orbit dot and highlight accents — never as a text color on white.
✓
Apply --r-full (pill) radius for all primary action buttons. Use --r-lg for cards and containers.
✕
Don't
✕
Never replace the gradient with a flat single color fill — not even the brand colors individually.
✕
Never rotate, skew, stretch, or add drop shadows to the mark. The orbit arc angle is fixed at −40°.
✕
Don't place the mark on busy backgrounds or images without a clear zone. Ensure contrast ≥ 3:1.
✕
Don't use more than two accent colors on a single surface. Lavender and Golden together — not all five.
✕
Never use system fonts or Arial in any designed surface. Montserrat must be loaded for all UI text.
